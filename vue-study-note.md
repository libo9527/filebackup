#### Vue 原生表单练习

给定一个表单，包含三个字段，其中两个是必填项。

```html
<form id="app" @submit="checkForm" action="https://vuejs.org/" method="post">
```

action 和 method 是必填项。

@submit 是 v-on:submit 的简写
